# 个人博客网站

这是一个简洁、现代的个人博客网站项目。该网站采用响应式设计，提供良好的用户体验和清晰的导航结构。

## 项目特点

- 简洁优雅的设计风格
- 响应式布局，适配各种设备
- 平滑滚动和动画效果
- 自动日期显示功能
- 智能导航栏（滚动时自动隐藏/显示）
- 博客文章淡入效果

## 项目结构

```
Web_start/
│
├── index.html      # 主页面
├── css/
│   └── style.css   # 样式表
├── js/
│   └── main.js     # JavaScript功能实现
└── README.txt      # 项目说明文档
```

## 技术栈

- HTML5
- CSS3
- 原生JavaScript
- 响应式设计
- 交互动画

## 主要功能

1. 导航菜单
   - 首页
   - 文章
   - 关于我
   - 联系方式

2. 博客特性
   - 简洁的设计风格
   - 响应式布局
   - 易于阅读的排版
   - 清晰的导航结构

3. 交互效果
   - 平滑滚动
   - 导航栏智能显示/隐藏
   - 文章卡片悬停效果
   - 内容淡入动画

## 如何使用

1. 克隆或下载项目到本地
2. 使用网页浏览器打开 index.html 文件
3. 开始浏览网站内容

- 修改样式：编辑 css/style.css 文件
- 添加新功能：在 js/main.js 中实现新的功能
